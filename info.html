Seminarier

Här kan du läsa mer om de olika seminarierna vi kommer erbjuda under konferensen:

En introduktion till HTML5 och dess nya funktioner

I detta seminarium kommer vi att ge en introduktion till HTML5, den senaste versionen av HTML-standarden. Vi kommer att gå igenom grunderna i HTML5 och dess syntax, samt fokusera på dess nya funktioner som gör det möjligt att skapa mer avancerade och interaktiva webbplatser. Du kommer också att få lära dig hur du använder HTML5-taggar som video, audio och canvas för att lägga till multimedia och ritverktyg på din webbplats. Detta seminarium är perfekt för dig som är nybörjare inom HTML5 och vill lära dig grunderna.

CSS3-tekniker för avancerad layout och animering

"CSS3-tekniker för avancerad layout och animering" är ett seminarium som fokuserar på hur man kan använda CSS3 för att skapa avancerade layouter och animeringar för webbsidor. Seminariet kommer att ge en djupgående introduktion till olika CSS3-tekniker och hur man kan använda dem för att skapa professionella och tilltalande webbsidor. Deltagarna kommer att lära sig om olika layouttekniker, hur man skapar animeringar och effekter med CSS3, och mycket mer. Seminariet riktar sig till personer som vill lära sig mer om CSS3 och hur man kan använda det för att skapa avancerade webbsidor

Responsiv design: hur du skapar användarvänliga webbplatser för alla enheter

"Responsiv design: hur du skapar användarvänliga webbplatser för alla enheter" är ett seminarium som fokuserar på hur man kan skapa webbplatser som är användarvänliga och lättanvända på alla enheter, inklusive bärbara datorer, surfplattor och smartphones. Seminariet kommer att ge en djupgående introduktion till tekniker för responsiv design, inklusive hur man använder olika layouttekniker, responsiva bilder och mediaquerys för att skapa webbplatser som ser bra ut och fungerar väl på alla enheter. Deltagarna kommer också att lära sig om hur man optimerar webbplatser för olika enheter och hur man kan använda verktyg för att testa och förbättra webbplatsernas användarvänlighet. Seminariet riktar sig till personer som vill lära sig mer om responsiv design och hur man kan skapa webbplatser som är användarvänliga på alla enheter.

JavaScript för nybörjare: grunderna för att skapa interaktiva webbplatser

"JavaScript för nybörjare: grunderna för att skapa interaktiva webbplatser" är ett seminarium som vänder sig till personer som är nya inom området JavaScript och vill lära sig grunderna för att skapa interaktiva webbplatser. Seminariet kommer att ge en grundläggande introduktion till JavaScript, inklusive hur man skriver kod, hur man använder variabler och funktioner, och hur man använder olika JavaScript-bibliotek för att skapa avancerade interaktiva webbplatser. Deltagarna kommer också att få möjlighet att testa och experimentera med olika tekniker för att skapa enklare webbplatser med JavaScript. Seminariet riktar sig till personer som är nyfikna på JavaScript och vill lära sig grunderna för att kunna skapa interaktiva webbplatser.

Modernt frontend-ramverk: en jämförelse mellan React, Angular och Vue

"Modernt frontend-ramverk: en jämförelse mellan React, Angular och Vue" är ett seminarium som fokuserar på de tre populäraste frontend-ramverken för webbutveckling: React, Angular och Vue. Seminariet kommer att ge en jämförelse mellan dessa tre ramverk, inklusive deras olika funktioner och fördelar. Deltagarna kommer att få en inblick i hur man använder var och en av dessa ramverk, och hur man kan välja det rätta ramverket för en given webbutvecklingsuppgift. Seminariet riktar sig till personer som är intresserade av frontend-ramverk och vill lära sig mer om de tre populäraste ramverken på marknaden.

En guide till frontend-säkerhet: hur du skyddar din webbplats från attacker

"En guide till frontend-säkerhet: hur du skyddar din webbplats från attacker" är ett seminarium som fokuserar på hur man kan skydda sin webbplats från olika typer av attacker, inklusive Cross-Site Scripting (XSS) och Cross-Site Request Forgery (CSRF). Seminariet kommer att ge en djupgående introduktion till olika tekniker och metoder för frontend-säkerhet, inklusive hur man använder säkerhetskopior, hur man skyddar känslig information och hur man hanterar användarbehörigheter på ett säkert sätt. Deltagarna kommer också att få lära sig om olika säkerhetsverktyg och hur man kan använda dem för att skydda sin webbplats. Seminariet riktar sig till personer som är intresserade av frontend-säkerhet och vill lära sig mer om hur man kan skydda sin webbplats från attacker.

Optimera dina webbplatser för prestanda: tips och verktyg för att öka hastigheten

"Optimera dina webbplatser för prestanda: tips och verktyg för att öka hastigheten" är ett seminarium som fokuserar på hur man kan optimera sin webbplats för bästa möjliga prestanda och hastighet. Seminariet kommer att ge en djupgående introduktion till olika tekniker och metoder för prestandaoptimering, inklusive hur man minifierar kod, hur man använder caching och hur man kan använda verktyg för att mäta och förbättra prestandan på sin webbplats. Deltagarna kommer också att få lära sig om olika prestandaverktyg och hur man kan använda dem för att hitta och åtgärda prestandaproblem på sin webbplats. Seminariet riktar sig till personer som är intresserade av prestandaoptimering och vill lära sig mer om hur man kan öka hastigheten på sin webbplats.

Så här använder du Web Components för att bygga återanvändbara komponenter

Seminariet "Så här använder du Web Components för att bygga återanvändbara komponenter" handlar om hur man kan använda tekniken Web Components för att skapa återanvändbara komponenter för webbplatser och webbapplikationer. Web Components är en samling av tekniker som gör det möjligt för utvecklare att skapa självständiga, anpassningsbara och återanvändbara komponenter som kan användas på olika webbplatser och webbappar. Seminariet kommer att täcka grunderna i Web Components och ge exempel på hur man kan använda tekniken för att bygga återanvändbara komponenter.

Öka din produktivitet med hjälp av verktyg som Babel och Webpack

Seminariet "Öka din produktivitet med hjälp av verktyg som Babel och Webpack" handlar om hur man kan använda verktyg som Babel och Webpack för att öka sin produktivitet som webbutvecklare. Babel är ett verktyg som gör det möjligt för utvecklare att skriva kod i modern JavaScript och sedan konvertera den till kompatibel kod som kan köras på äldre webbläsare. Webpack är ett verktyg som hjälper till att samla och packa ihop alla olika komponenter och resurser som behövs för att bygga en webbapplikation. Seminariet kommer att gå igenom hur man kan använda dessa verktyg för att öka sin produktivitet och göra det enklare att bygga komplexa webbappar.

En introduktion till designsystem för att skapa enhetlig och användarvänlig upplevelse

Seminariet "En introduktion till designsystem" fokuserar på hur man använder sig av ett designsystem för att skapa enhetlig och användarvänlig upplevelse i sina digitala produkter. Seminariet kommer att ge en översikt över vad ett designsystem är, hur det kan användas för att säkerställa en konsekvent design i olika digitala produkter, och hur det kan bidra till att skapa en bättre användarupplevelse. Deltagarna kommer att få lära sig om olika delar av ett designsystem, inklusive grundläggande designprinciper, färgscheman, typografi och komponenter. Seminariet riktar sig till personer som vill lära sig mer om hur man använder sig av designsystem för att skapa en enhetlig och användarvänlig upplevelse i sina digitala produkter.
